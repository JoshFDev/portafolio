¿Qué es este proyecto?

Sistema web fullstack para la administración completa de un club deportivo. Permite gestionar miembros, entrenadores, disciplinas, inscripciones y eventos deportivos desde una sola plataforma. Cuenta con autenticación por roles (administrador y miembro), recuperación de contraseña, y un panel diferenciado según el tipo de usuario. Es el proyecto más robusto y colaborativo que hemos desarrollado como equipo.

Módulos del sistema

El sistema cubre el ciclo completo de operación de un club deportivo:

Autenticación Login, roles y recuperación de contraseña
Gestión de Usuarios Creación y administración de cuentas
Miembros Alta, consulta y perfil de miembros
Entrenadores Registro y asignación por disciplina
Disciplinas Fútbol, baloncesto, taekwondo y más
Inscripciones Registro de miembros a disciplinas
Eventos Deportivos Creación y participación en eventos
Panel Admin Control total del sistema

Trabajo en equipo

Este sistema fue el resultado de un esfuerzo colaborativo real: dividimos los módulos por área de responsabilidad, usamos Git para coordinar el trabajo y resolvimos conflictos de integración entre el frontend en Angular y la API en Node.js. Aprendimos a comunicar decisiones técnicas, documentar endpoints y mantener consistencia en la base de datos compartida. Fue el proyecto que más nos exigió como equipo y del que más orgullosos estamos.

Lo que aprendí

Fue mi primera experiencia real con Angular 17 y su ecosistema: componentes standalone, servicios, guards de rutas y manejo de formularios reactivos. En el backend aprendí a estructurar una API REST con Node.js, manejar autenticación con JWT por roles y conectar con MariaDB. También reforcé el diseño de bases de datos relacionales con múltiples tablas interconectadas: miembros, entrenadores, disciplinas, inscripciones y eventos deportivos.
