¿Qué es este proyecto?

Sistema web desarrollado durante mi práctica integrativa en la CFE Zona Sur Volcanes, pensado para administrar el pago de los instructores y dar seguimiento a los cursos que imparten dentro de la institución.

Función principal

La aplicación optimiza el cálculo del pago de instructores con base en la UMA (Unidad de Medida y Actualización) y las horas cumplidas en cada curso, agilizando un proceso que antes se realizaba de forma manual y reduciendo el margen de error en los cálculos.

Stack tecnológico

▲ Angular Framework — capa de vista
JS Node.js Entorno de ejecución backend
EX Express Framework — API REST
DB MariaDB SQL Base de datos relacional

Lo que aprendí

Este proyecto me permitió trabajar con un stack completo (Angular, Node.js, Express y MariaDB) dentro de un entorno laboral real. Aprendí a automatizar cálculos administrativos sensibles, a diseñar una API REST con Express y a conectar el frontend en Angular con el backend de manera ordenada.
